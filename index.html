Скиллы

HTML

Язык разметки. Абсолютли все теги (те, которые не знаю, гуглю) знаю. Здесь даже учить ничего не надо, если есть скиллзы и понимания работы DOM.

CSS

Здесь немного посложнее. Продумывать дизайн, писать грамотный стиль, красиво анимировать элементы, адаптив для более примитивных гаджетов, для мобильных телефонов и т.д.

JS

Я люблю его, но он не очень понятный. Вернее понятный, но не везде. На этом сайте его заюзано не так много и не во всей его красе... Гораздо интереснее в приложениях

PHP

Прошлый век, тут его вообще не заюзано. Да и нахой оно не нужно, я делаю все на github pages. Тут он его не поддерживает, к сожалению

Sass

Sass очень удобный препроцессор CSS. Он позволяет сокращать код до минимума, у него есть наследование стилей, условия и т.д.

Bootstrap

Одна из самых удобных библиотек, позволяющих располагать элементы корректно, есть настройка адаптива.

Git

Гит и этим все сказано. Технология, которая предназначена для усовершенствования версий или отката в случае неудач и просто ностальгии...

NodeJS

NodeJS - платформа для серверных дел и для всего самого безупречного...

JQuery

Удобная библиотека, позволяющая работать с JS, сокращая код в разы, в десятки раз, в сотню раз...

Gulp

Автоматический сборщик всех файлов. Очень удобен при разработке. Мне даже он нравится больше, чем webpack....
